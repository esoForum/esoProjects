## esoForum for Projects
Using Projects (esoForum for Projects), anyone can create their own forum—taking on the same look and feel as esoForum—for a group, organisation, or project. **It's entirely free, forkable, and downloadable.**

Projects uses the esoTalk framework to deliver one's own esoForum, providing comfortable attractiveness while maintaining speedy performance.  Thanks to its simple structure, both maintaining and developing third-party addons for an esoForum 'project' are tasks accomplishable by the layperson.
